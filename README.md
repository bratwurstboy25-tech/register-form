# Landing Page Project

Простой современный лендинг с формой регистрации. Полностью адаптивный, с интерактивной формой и микроанимациями.

## Функционал

- Адаптивная верстка для мобильных и десктопных устройств
- Проверка имени и email с instant feedback
- Плавный скролл к форме
- Сообщение об успешной отправке формы
- Hover эффекты и микроанимации
- Чистый и структурированный код
