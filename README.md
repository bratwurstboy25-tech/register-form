# Landing Page Project

Простой современный лендинг с формой регистрации.

## Функционал

- Адаптивная вёрстка
- Проверка имени и email
- Instant feedback при вводе данных
- Плавный скролл к форме
- Сообщение об успешной отправке
- Hover эффекты и микроанимации

## Технологии

- HTML5, CSS3 (Flex/Grid, переменные)
- JavaScript (DOM, события, валидация)
